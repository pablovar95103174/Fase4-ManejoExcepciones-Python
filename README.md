# Descripción del Proyecto

Este proyecto fue desarrollado como parte de la Fase 4 del curso de Programación 213023 de la UNAD. El objetivo principal de la actividad es implementar manejo de excepciones en Python utilizando programación orientada a objetos, garantizando estabilidad, robustez y adecuado control de errores durante la ejecución de la aplicación.

Durante el desarrollo del proyecto se utilizaron estructuras de control de errores mediante los bloques `try`, `except` y `finally`, permitiendo evitar fallos inesperados y mejorar el comportamiento general del sistema.

Además, el trabajo fue realizado de forma colaborativa utilizando GitHub como herramienta para la administración y control del código fuente.

---

# Objetivos del Proyecto

## Objetivo General

Implementar manejo de excepciones en una aplicación desarrollada en Python para controlar errores y garantizar el correcto funcionamiento del sistema.

## Objetivos Específicos

- Aplicar estructuras de manejo de excepciones en Python.
- Implementar programación orientada a objetos.
- Controlar errores durante la ejecución del programa.
- Mejorar la estabilidad y robustez de la aplicación.
- Utilizar GitHub como herramienta colaborativa.
